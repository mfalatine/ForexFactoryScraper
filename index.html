🔌 API Usage

You can fetch this data directly in your applications using these URLs:

/.netlify/functions/scrape?start=YYYY-MM-DD
/.netlify/functions/scrape?start=YYYY-MM-DD&format=csv

Example Usage (JavaScript):

fetch('YOUR_URL_HERE')
  .then(response => response.json())
  .then(data => {
    console.log('Forex events:', data);
    // Use the data in your application
  });

Example Usage (Python):

import requests
import pandas as pd

# Get JSON data
response = requests.get('YOUR_URL_HERE')
data = response.json()

# Or get CSV directly into pandas
df = pd.read_csv('YOUR_URL_HERE')
